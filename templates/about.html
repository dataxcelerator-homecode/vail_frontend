About Keyboard Visualizer

A professional tool for visualizing Vial keyboard configurations

Features

  • 📊 Multi-layer visualization: See all your keyboard layers at once
  • 🎨 Color-coded keys: Easy identification of key types and functions
  • 🔄 Keycode transformation: Replace keycodes before visualization
  • 💾 High-quality output: PNG images suitable for documentation
  • ⚡ Fast processing: Generate visualizations in seconds
  • 🔐 Privacy-focused: All processing happens locally

Symbol Legend

Symbol Meaning
▽ Transparent key (KC_TRNS)
✗ No key (KC_NO)
⇧ Shift modifier
⌃ Control modifier
⌥ Alt/Option modifier
⌘ GUI/Command modifier

Usage Example

Command-line interface is also available:

# Basic usage
python cli.py input.vil output.png

# With keycode transformation
python cli.py input.vil output.png --rename-layer 4 --rename-old KC_TRNS --rename-new KC_NO

# Enable debug logging
python cli.py input.vil output.png --debug

Technology Stack

Python 3 Flask Matplotlib QMK/Vial
